To open the game in the web version:

- Download the BIRD.jsdos from the Releases section.

- Open this link: https://jonybonyy.github.io/JonyBird-web/

- Than press play and press on the button at the middle, insert the BIRD.jsdos file there and press play.

- Press esc to see what are all the game mechanics

Later I'll make it work automatically (maybe).

THE WEB VERSION ONLY WORKS IN PC FOR NOW

To play the game not on the web version:

- Go to the Releases section and press the + 1 Releas

- Download the Bird.exe file

- Download DOSBox: https://www.dosbox.com/download.php?main=1

- And finally, run it there.
